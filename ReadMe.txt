Todas bibliotecas utilizadas são padrão do python.

Para executar o programa, basta executar o seguinte comando no diretório raiz do projeto:

python3 Main.py <caminho processes.txt> <caminho files.txt> <caminho string.txt>

Onde os 3 argumentos de caminho são o caminho, absoluto ou relativo, até o documento específico do caso de teste.
